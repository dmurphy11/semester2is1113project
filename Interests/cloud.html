SaaS - Salesforce

Saas stands for software as a service, and is a way of delivering centrally hosted applications over the Internet, as a service. Instead of installing and maintaining software, you simply access it via the Internet, freeing yourself from complex software and hardware management. There are many advantages to SaaS, none more so than the low initial costs and seamless upgrades, minimizing needless stress for customers. SaaS customers have no hardware or software to buy, install, maintain, or update. Access to applications is easy, you just need an Internet connection. Customisation is also a benefit of SaaS. Users can customise applications to fit their business model and needs/desires. These unique customisations are always preserved through upgrades, accommodating less customer risk and low adoption costs with upgrades.

PaaS - Cloud 9

Owned by Amazon, Cloud9 is a cloud-based integrated development environment that lets you write, run, and debug your code with just a browser. It includes all the tools a user would need for web development, including a code editor, debugger and terminal. C9 is also compatible with all major programming languages including JavaScript, PHP and Python to name but a few. C9 can also be linked to Github, a web-based hosting service, and Heroku, a web application deployment model. Not only are all of these services available free of charge, they were very useful for me personally during the development of this project, with C9 acting as the development platform and Heroku providing particularly useful in runninng the application to see how the finished product would look. The live preview feature on C9 was also an efficient way to monitor potential changes to the project. In my eyes, one would find it hard to find a web-development platform as stress free and easy to work with as Cloud 9!

IaaS - Amazon

Amazon web services (AWS), a subsidary of Amazon, provides many computing and networking products/services. During this project I have been able to exploit one of these products, Cloud9. They also have services like Amazon Elastic Beanstalk, which is the equivalent to the Heroku platform which we use, so Amazon really have all bases covered. Amazon launched AWS in July 2002, when the initial revenue generator was amazon selling access to virtual servers as a service. What AWS now offer is a secure cloud services platform, data storage and all the tools necessary to allow ones business grow and progress. AWS now has over 100 products and services to its name, from virtual reality to exploiting the very successful "Alexa" for Business. Many apps which young people would be very familiar with, Spotify, Shazam and Soundcloud for example, all use AWS to expand storage (Spotify) and deal with spike times (Shazam during the Superbowl). According to the most recent statistics, AWS operates an estimated 1.4 million servers. It is clear Amazon is clearly a lot more than the buy and sell site I once thought it was. The more research one does into it, they discover companies and products that are run by Amazon, which they would least expect!

CaaS - Gmail

Gmail, an Email service run by Google, is one of the leading email service providers in the world. Not only is it free, users get 15 gigabytes of storage, can receive emails up to 50MB in size and also have access to a range of features like a personal planner, googles version of Microsoft Office and, in my opinion the most useful, Google Drive, a cloud based file storage application, which I found particularly useful when working on a group project recently. We were all able to access the project by logging into google drive, and not having to physically meet and pass on a USB or a CD. Each student here at University College Cork gets a Gmail address synched with our student number and used to access all of the IT services available. Gmail also has a mobile app so one will never miss an important email such as a cancelled lecture. I had previously had a hotmail (Outlook) email address but after using Gmail I would never go back to another email provider. Personally I think Gmail is the best available and I can see it going from strength to strength in the future and becoming stronger than it already is.
